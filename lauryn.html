My Projects
FastPort

In this project my group decided to design and create a third party online website dedicated to renewing and applying for Id and Passports online. The concept behind this project is to help users easily apply and navigate this website without having to go to governement offices in person.

Honey Beeuty

In this project me and my team mates were tasked with designing and creating a fully functional wireframe for a fake business. In this project we based our project around a honey based online beauty store. The wireframe was fully prototyped to complete the transaction process.

D3 games was the first mid-term project for term 1 of Web development class. In this project, my group and I decided to designa and code a semi working games catalouge, similar to Steam. The designs of the login and calalouge page were both designed by me.
